#### Claims that need stronger public evidence before using as neutral product facts

- Any specific uplift claims such as recovery percentages, cancellation-reduction percentages, LTV multiples, CSAT scores, or customer case-study results should be labeled as vendor/customer-marketing claims unless independently verified.
- Enterprise security features such as SSO, SCIM, data residency, customer-managed keys, or formal SLAs should not be asserted without current documentation or vendor confirmation.
